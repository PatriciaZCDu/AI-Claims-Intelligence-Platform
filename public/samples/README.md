# Sample damage photos

Drop 2–4 JPEGs of damaged vehicles here (e.g. `front-damage.jpg`, `rear-damage.jpg`)
to demo live uploads on the **Create Claim** screen.

The platform does not require them: in the default `mock` AI mode the pipeline
produces a deterministic assessment with or without images, and the seeded
claims already include rich findings. Add real photos here only when you want to
exercise a live vision provider (Gemini / Claude / OpenAI) end-to-end.

Any standard car-damage image works. Keep the short edge ≥ 500px so the
resolution validation check passes.
